# eea.rabbitmq.plone

Configuration and utilities for RabbitMQ client

Install by adding it to your buildout::

    [buildout]

    ...

    eggs =
        eea.rabbitmq_plone


and then running ``bin/buildout``

For testing:

    /plone/instance# ./bin/test -s eea.rabbitmq_plone
